== Description ==

WP Jet Player for WordPress enables you to publish videos on your posts, pages and widgets using the user friendly video player on the web. Take complete control of your player, from branding to size and dimensions.

= WP Jet Player Features =
Here are the most notable features of WP Jet Player plugin.

* Add multiple videos on same page or post.
* Customize player icons and video thumbnail image.
* Take screenshot while playing video.
* Hide Subtitle of video.
* Subtitle Offset time.
* Flip video screen Normal, Horizontal, Vertical.
* Video Play speed control.
* Aspect Ratio.
* Multiple Screen size options (Web Full Screen, mini or sticky player, and full window screen).
* Full Responsive Player.
* Video Qualities user HD, MP4 etc.

Note: only Self hosted videos are supported now.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type WP Jet Player
3. Find WP Jet Player and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
